# 🚀 Expo Go 启动命令

## 快速启动

### 方法一：使用启动脚本（最简单）

双击运行 `启动ExpoGo.bat`，然后选择启动模式。

### 方法二：使用命令行

#### LAN 模式（推荐，手机和电脑需同一 WiFi）


```

或：

```bash
cd F:\A\mobile-app
npx expo start --lan
```

#### 隧道模式（最稳定，不依赖局域网）

```bash
cd F:\A\mobile-app
npm run start:tunnel
```

首次使用需要登录：
```bash
npx expo login
```

#### 清除缓存启动

```bash
cd F:\A\mobile-app
npm run start:clear
```

## 📱 连接 Expo Go

启动后：

1. **打开 Expo Go 应用**
2. **扫描二维码**（终端会显示）
3. **或手动输入 URL**（终端显示的地址）

## 🔧 如果连接失败

### 问题：Failed to download remote update

**解决方案**：

1. **使用隧道模式**（最可靠）：
   ```bash
   npm run start:tunnel
   ```

2. **检查网络**：
   - 确保手机和电脑在同一 WiFi
   - 检查防火墙设置

3. **清除缓存**：
   ```bash
   npm run start:clear
   ```

## 💡 推荐

**最简单的方法**：使用隧道模式
```bash
npm run start:tunnel
```

这样就不需要担心网络配置问题了！
